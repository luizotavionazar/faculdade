Tabelas:

Clientes ()
id_cliente int ai nn uq (PK)
nome varchar(45) nn
cpf varchar(11) nn uq
telefone varchar(11) nn
endereco varchar(60) 

Equipamentos (Equipamentos)
Contém os equipamentos disponíveis para aluguel.

id_equipamento int ai nn uq (PK)
descricao varchar(60) nn
preco_diaria double nn
quantidade_total int nn
quantidade_disponivel int nn

Contratos (Contratos)
Registra as informações sobre cada aluguel, como o cliente, o equipamento alugado e o período de locação.

id_contrato int ai nn uq (PK) 
id_cliente int nn (FK)
id_equipamento int nn (FK)
data_inicio datetime nn
data_fim datetime nn
valor_total float nn 
status_contrato (ex: ativo, finalizado, cancelado) varchar(1) nn

Relacionamentos:
Clientes x Contratos: Um cliente pode ter vários contratos, mas cada contrato pertence a um único cliente.
Equipamentos x Contratos: Um contrato envolve um equipamento, mas cada equipamento pode estar em vários contratos.
