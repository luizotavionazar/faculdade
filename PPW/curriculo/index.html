Mini Curso Scratch

Participei como instrutor em um projeto de extensão voltado para o ensino de programação em blocos utilizando o Scratch. O objetivo foi introduzir conceitos fundamentais de programação de maneira acessível e interativa, especialmente para iniciantes e jovens estudantes.


Durante o curso, os participantes aprenderam sobre lógica de programação, estruturas de repetição, eventos e variáveis, explorando esses conceitos por meio da criação de histórias, animações e jogos. A abordagem prática e visual do Scratch permitiu que os alunos desenvolvessem não apenas habilidades técnicas, mas também criatividade e pensamento computacional.


O projeto teve um impacto significativo na democratização do ensino de programação, tornando-o mais atrativo e acessível para diferentes públicos. O engajamento dos alunos ao longo das atividades reforçou a importância de metodologias lúdicas para despertar o interesse por tecnologia e inovação.

Loja Online - Kometaz

Link do projeto

A Loja Online Kometaz é um projeto de e-commerce desenvolvido como parte do meu aprendizado em desenvolvimento web. A plataforma foi construída utilizando HTML e CSS, com foco na criação de uma interface clara e organizada para facilitar a navegação dos usuários.

O site conta com as seguintes funcionalidades:

  • Página inicial: Apresenta um banner chamativo e um menu de navegação simples, com links para as seções principais (Home, Sobre e Produtos).

  • Catálogo de produtos: Os produtos são organizados em categorias, como Eletrônicos, Casa, Vestuário, Decoração e Pet Shop, facilitando a busca pelos itens desejados.

  • Páginas de produtos: Cada produto possui uma página dedicada com informações detalhadas, como descrição, preço à vista, opções de parcelamento e botão de compra.

  • Seção "Sobre": Uma página que explica a missão e os valores da Kometaz, criando uma conexão com o cliente.

Embora o site não seja responsivo e não utilize JavaScript, ele foi desenvolvido com foco na simplicidade e na usabilidade, priorizando uma experiência de navegação clara e direta. O design foi pensado para ser visualmente atraente, com cores e elementos que destacam as informações mais importantes.

Este projeto foi uma ótima oportunidade para praticar e aprimorar minhas habilidades em HTML e CSS, além de entender melhor os desafios de criar uma loja virtual funcional e organizada.

Plataforma de Reprodução de Músicas

Link do projeto

A Plataforma de Reprodução de Músicas é um projeto inspirado no Spotify, desenvolvido para simular uma experiência de streaming de áudio. A plataforma foi criada utilizando HTML e CSS, com foco em um design moderno e intuitivo, que permite aos usuários navegar por playlists, artistas e músicas de forma fácil e agradável.


O projeto inclui as seguintes funcionalidades:

  • Menu de navegação: Com botões para voltar, avançar, acessar a página inicial e realizar buscas.
  • Menu lateral: Exibe playlists, músicas curtidas e álbuns salvos, com ícones e imagens dos artistas.
  • Página de playlist: Apresenta uma capa, informações detalhadas (como nome da playlist, descrição e duração) e uma lista de músicas com capas dos álbuns, títulos, artistas e duração.
  • Player de música: Controles de reprodução (play, pause, próximo, anterior), barra de progresso e ajuste de volume.

O design foi pensado para ser visualmente atraente, com cores e elementos que remetem ao estilo do Spotify. Apesar de não utilizar JavaScript, a plataforma oferece uma experiência de navegação fluida e organizada, com foco na simplicidade e na usabilidade.

Este projeto foi uma ótima oportunidade para praticar e aprimorar minhas habilidades em HTML e CSS, além de entender melhor os desafios de criar uma interface de streaming de áudio.

Ataque Maluco - Jogo de Ataque em Java

Link do projeto

O Ataque Maluco é um jogo de turnos desenvolvido em Java, onde dois jogadores se enfrentam em uma batalha estratégica. O objetivo do jogo é reduzir a energia do oponente a zero, utilizando ataques com diferentes níveis de poder e efeitos especiais.

O jogo foi criado com foco em programação orientada a objetos (POO) e utiliza interfaces gráficas simples através da biblioteca Swing. As principais funcionalidades incluem:

  • Sistema de turnos: Cada jogador ataca alternadamente, com chances de acertar ataques fracos, fortes ou pancadas.

  • Ataques especiais: Pancadas podem causar dano extra ou curar o atacante, dependendo da sorte.

  • Interface gráfica: Janelas interativas exibem o status do jogo, resultados dos ataques e opções para os jogadores.

  • Desistência: Um jogador pode desistir da partida se estiver em desvantagem, encerrando o jogo.

O projeto foi desenvolvido para praticar conceitos avançados de Java, como manipulação de exceções, uso de bibliotecas gráficas e lógica de programação para simular batalhas por turnos. Além disso, o jogo possui um sistema de probabilidades que torna cada partida única e imprevisível.

Este projeto foi uma ótima oportunidade para aprimorar minhas habilidades em Java e entender melhor como criar interfaces gráficas simples e funcionais.
